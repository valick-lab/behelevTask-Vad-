В этом проекте сделана простая библиотека калькулятора и консольное приложение.



Библиотека (`CalculatorLib`) содержит методы:

\- сложение

\- вычитание

\- умножение

\- деление



Консольное приложение (`CalculatorApp`) подключает эту библиотеку и вызывает её методы.



Программа создаёт объект калькулятора и выполняет простые математические операции с числами, выводя результат в консоль.
